# TypesScript Boilerplate for browser

A boring project template for browser.

## Getting started

### Check environment

```sh
$ npm -v
10.1.0
$ node --version
v20.6.1
```

### Start development server

```
$ npm run dev
```

### Create bundle

```
$ npm run build
```

### Create bundle (when the sources has changed)

```
$ npm run watch
```
