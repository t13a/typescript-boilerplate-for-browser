# TypesScript Boilerplate for browser

A boring project template for browser.

## Getting started

### Check environment

```sh
$ npm -v
10.1.0
$ node --version
v20.6.1
```

### Install packages

```
$ npm ci
```

### Start development server

```
$ npm run dev
```

### Create bundle

```
$ npm run build
$ npm run watch # Build when files have been changed.
```

### Linting and formatting

```
$ npm run lint
$ npm run fmt
```
